프로젝트 개요

본 시스템은 스마트 팩토리 인프라 최적화를 목표로, 다중 워크스테이션 환경을 협동 로봇이 자율 순회하며 부품 상태를 정밀 진단하는 하이엔드 무인 검사 솔루션입니다.

로봇 엔지니어링, AI 오브젝트 디텍션(YOLOv11x) 및 실시간 3D 기하학 분석 처리를 통합하여, 수집된 제조 빅데이터를 디지털 트윈으로 구현된 대시보드를 통해 완벽하게 제어 관리할 수 있도록 설계되었습니다.

트러블슈팅 (Troubleshooting)

■ Issue 01. 3D 연속 스트리밍에 따른 웹 가동 프레임 드랍 및 병목 현상
발생 문제
3D 점구름 데이터를 지속적이고 연속적으로 전송/갱신을 시도함에 따라 브라우저 메모리 부하와 데이터베이스 상태 비동기 처리가 밀려 웹 관제 화면에 극심한 지연(렉) 현상 봉착.
해결 방안
지속 전송 방식을 폐기하고 로봇의 스캔 임무 완료 응답(3d_scan_done=True) 직후 시점에만 정제된 데이터를 가져오는 '캡처 기반 로드 시스템'을 도입. 더불어 배경 점구름 크기를 조절하는 가변 샘플링 필터를 주입하여 성능 제어력 확보.
데이터 샘플링 스케일 및 캡처형 로드 UI 디자인 (sampling_control_ui.png)
■ Issue 02. 초기 화면 축 뒤틀림 현상 및 마우스 회전 제어 오류
발생 문제
웹 3D 뷰어 컴포넌트 호출 시 정밀 입체 각도가 아닌 2D 평면과 유사하게 축이 고정 출력되었고, 마우스 드래그 조작이 뷰포트 회전이 아닌 화면 평면 이동(Pan)으로 오작동하여 입체 형상 검수 방해.
해결 방안
데이터 파싱 과정에서 type: 'scatter3d' 차원 정보를 명시적으로 확정하고 레이아웃의 디폴트 액션을 회전 모드(dragmode: 'rotate')로 강제 지정. 초기 카메라 뷰를 45도 상단 대각선 탑다운 뷰포트(scene.camera.eye)로 맞춰 완벽한 직관성 구현.
3D 공간 뷰포트 왜곡 수정 및 고정 앵글 셋업 (3d_viewport_success.png)
■ Issue 03. 복잡한 배경 데이터 내 불량 식별성 저하 및 오인지 리스크
발생 문제
촘촘하게 군집을 이룬 3D 배경 점구름들 사이에 AI가 탐지한 핵심 결함 알림 마커 및 정보 라벨 폰트 크기가 파묻혀 가혹한 현장 조명이 깔린 모니터링 환경에서 가독성이 심각하게 저하됨.
해결 방안
현장 중심 가독성 개선을 위해 결함 마커의 고유 크기 파라미터를 대폭 확장 조정하고(MARKER_SIZE=18), 가시성이 극대화된 텍스트 크기와 강한 보색 캘리브레이션을 결합함으로써 현장 오퍼레이터가 한눈에 결함을 판독 가능하도록 인터페이스 강화.
오퍼레이터 시인성 개선용 마커 및 라벨 스케일 스크린샷 (operator_view_calibration.png)
